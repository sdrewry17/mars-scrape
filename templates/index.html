Mars Analysis

Mars is an interesting place!

This was a useful project showcasing various tools to collect, store, and return live data to a webpage.

  • Python - Program used to control tasks, using multiple libraries
  • Beautiful Soup - Python library used to parse html data
  • Splinter - Python library used to control a web browser and navigate to different parts of websites
  • Tweepy - Python API wrapper to pull data from the Twitter API
  • Pandas - Python library used to read an HTML page into a DataFrame
  • Flask - Python library used to connect html to python objects
  • MongoDB - NoSQL database
  • PyMongo - Python connector for MongoDB
  • Flask PyMongo - Python connector MongoDB using Flask
